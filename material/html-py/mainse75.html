A.6 Gráficos

Para criar um esboço do gráfico de uma função de uma variável real y = f(x), podemos usar a biblioteca Python mathplotlib.A função matplotlib.pyplot.plot faz uma representação gráfica de um conjunto de pontos {(xi,yi)} fornecidos. Existe uma série de opções para esta função de forma que o usuário pode ajustar várias questões de visualização. Veja a documentação.

Exemplo A.6.1. Veja as seguintes linhas de código: 

>>> import numpy as np
>>> import matplotlib.pyplot as plt
>>> def f(x): return x**3 + 1
...
>>> x = np.linspace(-2,2)
>>> plt.plot(x, f(x))
[<matplotlib.lines.Line2D object at 0x7f4f6d153510>]
>>> plt.grid()
>>> plt.show()
