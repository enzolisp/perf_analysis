Capítulo 8
Derivação numérica

Nesta seção, trataremos das estratégias numéricas para aproximação de derivadas de funções reais. Com as técnicas abordadas, é possível calcular aproximadamente a derivada de uma função a partir de um conjunto discreto de pontos {(xi,yi)}i=1n. Começamos discutindo as chamadas aproximações por diferenças finitas e, então, as aproximações de derivadas via ajuste ou interpolação.

Ao longo deste capítulo, assumiremos que as seguintes bibliotecas e módulos Python estão importados:

import matplotlib.pyplot as plt
