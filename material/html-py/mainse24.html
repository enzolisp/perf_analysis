4.8 Cálculo de autovalores e autovetores

Considere o problema de autovalores Av = λv, onde A é uma matriz diagonalizável, isto é, existe uma matriz diagonal D e uma matriz ortogonal U tal que A = UDU-1.

4.8.1 Método da potência

O método da potência para cálculo do maior autovalor (em módulo) consiste na iteração

{ x(1) = aprox. inicial do autovetor, λ(1) = x(1)T Ax(1), { x(k+1) = Ax(k) ∥Ax(k)∥2, λ(k+1) = x(k+1)T Ax(k+1), ,k ≥ 1.

(4.280)

Observação 4.8.1. Observe que na iteração do método da potência (4.280), temos Ax(k) = A(k)x(1).

Exemplo 4.8.1. A seguinte matriz

A = [ 1 0 0 2 3 0 3 4 2 ]

(4.281)

tem λ = 3 como maior autovalor (por quê?). Tomando x(1) = (1, 1, 1), a iteração do método da potência nos fornece os seguintes resultados:

k x(k) λ(k)
1 (1, 1, 1) 15
2 (0,08,0,41,0,91) 4,10
3 (0,02,0,34,0,94) 3,50
4 (0,01,0,30,0,95) 3,28
5 (0,00,0,28,0,96) 3,16
⋮ ⋮ ⋮
14 (0,00,0,24,0,97) 3,00

Código Python: Método das Potências
from __future__ import division import numpy as np def metodo_potencias(A, x0, tol=1e-10, it_max=1000):     x = np.copy(x0)     p = np.argmax(x)     x = x/x[p]     for i in range(it_max):         y = np.dot(A, x)         u = y[p]  # lambda = autovalor         p = np.argmax(y)         if y[p] == 0:             return 0, x         x1 = y/y[p]         if np.abs(x - x1).max() < tol:  # verificando a tolerancia             return u, x1         x = x1     raise NameError('num. max. de iteracoes excedido.') A = np.array([         [1, 0, 0],         [2, 3, 0],         [3, 4, 2]         ]) x0 = np.ones((3, 1))  # Aproximacao inicial u, x = metodo_potencias(A, x0) print(u) print(x.T)

Para entendermos melhor o comportamento assintótico da sequência {x(n)} n≥1, primeiro consideramos o caso particular onde A é uma matriz diagonal, isto é,

A = [ λ1 0 0 ⋯ 0 0 λ2 0 ⋯ 0 0 0 λ3 ⋯ 0 ⋮ ⋱ 0 0 0 ⋯ λn ] .

(4.282)

Suponha que um dos autovalores seja estritamente maior que os demais, isto é, |λ1| > |λ2|≥|λ3|≥⋯ ≥|λn|. Dado x(1) = (ξ 1,ξ2,ξ3,…,ξn), então

Akx(1) = Ak [ ξ1 ξ2 ξ3 ⋮ ξ n ] = [ λ1kξ 1 λ2kξ 2 λ3kξ 3 ⋮ λnkξ n ] = λ1kξ 1 [ 1 ξ2 ξ1 (λ2 λ1 ) k ξ3 ξ1 (λ3 λ1 ) k ⋮ ξn ξ1 (λn λ1 ) k ] ,

(4.283)

desde que ξ1≠0. Como λn λ1 ≤ λn-1 λ1 ≤⋯λ3 λ1 ≤ λ2 λ1 < 1, então (λj λ1 ) k tende a 0 para cada j, 2 ≤ j ≤ n. Devido à normalização realizada em cada passo da sequência,

x(k+1) = Akx(1) ∥Akx(1)∥2

(4.284)

converge para ± e1, e1 = (1, 0, 0,…, 0). Também, a sequência

λ(k) = x(k)T Ax(k)

(4.285)

converge para λ1, pois

lim ⁡ k→∞λ(k) = (±e 1)T A(±e 1) = λ1e1T e 1 = λ1.

(4.286)

Considere, agora, o caso onde A é diagonalizável, ou seja, A = UDU-1 com U uma matriz ortogonal contendo os autovetores em cada coluna e D uma matriz diagonal contendo os autovalores:

D = [ λ1 0 0 ⋯ 0 0 λ2 0 ⋯ 0 0 0 λ3 ⋯ 0 ⋮ ⋱ 0 0 0 ⋯ λn ] .

(4.287)

Considere a mesma hipótese sobre os autovalores: |λ1| > |λ2|≥|λ3|≥⋯ ≥|λn|. Então

Ak = (UDU-1)(UDU-1)(UDU-1)⋯(UDU-1) = UDkU-1

(4.288)

visto que UU-1 = I. Suponha que o chute inicial x(1) pode ser escrito da forma

x(1) = UU-1x(1) = U[ξ 1ξ2ξ3⋯ξn]T

(4.289)

com ξ1≠0. Então

Akx(1) = (UDkU-1)U [ ξ1 ξ2 ξ3 ⋮ ξ n ] = U [ λ1kξ 1 λ2kξ 2 λ3kξ 3 ⋮ λnkξ n ] = λ1kξ 1U [ 1 ξ2 ξ1 (λ2 λ1 ) k ξ3 ξ1 (λ3 λ1 ) k ⋮ ξn ξ1 (λn λ1 ) k ] .

(4.290)

Como na discussão anterior, o último vetor converge para ± e1 e

x(k) = Akx(1) ∥Akx(1)∥2

(4.291)

converge para v1 = ±Ue1 que é um múltiplo do autovetor associado a λ1. Também, a sequência

λ(k) = x(k)T Ax(k)

(4.292)

converge para o autovalor dominante λ1:

lim ⁡ k→∞λ(k) = v 1T Av 1 = λ1v1T v 1 = λ1.

(4.293)

Observação 4.8.2. O método da potência tem duas restrições:

  • A aproximação inicial x(1) não pode ser ortogonal ao autovetor associado ao autovalor dominante.

  • Um autovalor deve ter o módulo estritamente maior que os demais. Essa restrição impede o funcionamento do método no caso em que o autovalor dominante é complexo, pois eles aparecem em pares conjugados, possuindo o mesmo módulo.

Outra análise para a convergência do método

Aqui, vamos apresentar uma análise alternativa para a convergência do método da potência: se A ∈ ℝn,n é diagonalizável, então existe um conjunto {vj}j=1n de autovetores de A tais que qualquer elemento x ∈ ℝn pode ser escrito como uma combinação linear dos vj. Sejam {λj}j=1n o conjunto de autovalores associados aos autovetores tal que um deles seja dominante, ou seja, |λ1| > |λ2|≥|λ3|≥⋯|λn| > 0. Como os autovetores são linearmente independentes, todo vetor x ∈ ℝn, x = (x1,x2,...,xn), pode ser escrito com combinação linear dos autovetores da seguinte forma:

x = ∑ j=1nβ jvj.

(4.294)

Observamos que se x está na forma (4.294), então Akx pode ser escrito como

Akx = ∑ j=1nβ jAkv j = ∑ j=1nβ jλjkv j = β1λ1k (v 1 + ∑ j=2nβj β1 (λj λ1 ) kv j) .

(4.295)

Como |λj λ1 | < 1 para todo j ≥ 2, temos

∑ j=2nβj β1 (λj λ1 ) kv j → 0.

(4.296)

Assim,

Akx ∥Akx∥2 = β1λ1k ∥Akx∥ (v1 + O ( |λ2 λ1 | k)) .

(4.297)

Como a norma de Akx ∥Akx∥ é igual a um, temos

∥β1λ1k ∥Akx∥v1∥ → 1

(4.298)

e, portanto,

|β1λ1k ∥Akx∥ | → 1 ∥v1∥ .

(4.299)

Ou seja, se definimos α(k) = β1λ1k ∥Akx∥ , então

|α(k)|→ 1.

(4.300)

Retornando a (4.297), temos:

Akx ∥Akx∥ - α(k)v 1 → 0.

(4.301)

Observe que um múltiplo de autovetor também é um autovetor e, portanto,

x(k) = Akx(1) ∥Akx(1)∥ .

(4.302)

é um esquema que oscila entre os autovetores ou converge para o autovetor v1.

4.8.2 Método da iteração inversa

Nesta seção, vamos estudar a sequência

x0, (A - σI)-1x 0 ∥(A - σI)-1x0∥2, (A - σI)-2x 0 ∥(A - σI)-2x0∥2, (A - σI)-3x 0 ∥(A - σI)-3x0∥2,⋯

(4.303)

para valores iniciais x0 e σ. Para o caso onde A é diagonalizável podemos escrever A = UDU-1 com D diagonal,

D = [ λ1 0 0 ⋯ 0 0 λ2 0 ⋯ 0 0 0 λ3 ⋯ 0 ⋮ ⋱ 0 0 0 ⋯ λn ] .

(4.304)

Assim, A - σI = U(D - σI)U-1 e, portanto, (A - σI)-1 = U(D - σI)-1U-1. Observamos que as matrizes A e (A - σI)-1 possuem os mesmos autovetores associados aos autovalores λj e (λj - σ)-1, respectivamente. Agora, supomos que σ satisfaça |λk - σ| < |λj - σ| para algum k e para todo j≠k. Também, Supomos que o chute inicial x0 possa ser escrito da forma

x0 = UU-1x 0 = U[ξ1ξ2ξ3⋯ξn]T

(4.305)

com ξk≠0. Então (A - σI)-kx 0 = (U(D - σI)-kU-1)U [ ξ1 ξ2 ξ3 ⋮ ξ n ] (4.306) = U [ (λ1 - σ)-kξ 1 (λ2 - σ)-kξ 2 (λ3 - σ)-kξ 3 ⋮ (λn - σ)-kξ n ] = (λi - σ)-kξ 1U [ ξ1 ξi ( λi-σ λ1-σ ) k ⋯ 1 ⋮ ⋯ ξn ξi ( λi-σ λn-σ ) k ] . (4.307)

Como |λi-σ λj-σ | < 1, o último vetor converge para ± ei e

xk = (A - σI)-kx 0 ∥(A - σI)-kx0∥2

(4.308)

converge para Uei = vi que é um múltiplo do autovetor associado a λi. Também, a sequência

λk = xkT Ax k

(4.309)

converge para λi:

lim ⁡ k→∞λk = viT Av i = λiviT v i = λi.

(4.310)

A método da iteração inversa tem restrições semelhantes àquelas do método da potência:

  • O chute x0 não pode ser ortogonal ao autovetor associado ao autovalor λi.

  • O chute σ deve estar mais próximo de λi do que dos λj, j≠i.

A vantagem é que conseguimos calcular qualquer autovalor, não apenas o autovalor dominante.
